# Подсказки по командной строке

Команда смены дериктории
 ```shcd
 c:\folder_name
 ```

Команда отображение текущей директории только для мак и линукса 
```sh
pwd
```
Листинг текущей дериктории для винды :
```sh
dir

```
Для Линукса и Мака:
```sh
ls
```
Удаление файла в винде :
```sh
del <filename>
```
В Линуксе и Маке :
```sh
rm <filename>
```
